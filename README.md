Технологии JS, React, Redux

1. Работа выполнена в классическом стиле, с использованием функциональных компонент;
2. Стейт, и все маниупляции с ним проходят через Redux;
3. Функционал приложения стандартный, дела можно добавлять, удалять, редактировать, и фильтровать.
4. Фичей является дизайн и анимации. Дела выстраиваются в сетку при помощи флекса.
5. Каждый элемент анимирован (при помощи библиотеки framer-motion) на появление и уход, также присутсвуют легкие анимации на ховер эффектах; 
6. Иконки импортированы из библиотеки react-icons;
